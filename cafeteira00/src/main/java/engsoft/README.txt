INF319 Projeto Orientado a Objetos: Cafeteira

Repita até obter uma primeira implementação funcional da Cafeira:
1. Estude a especificação fornecida e o código fonte das classes:
   EstadoHardware.java, Hardware.java e JanelaCafeteira.java
2. Analise, Projete e Implemente a Cafeira
   Todo código programado deve ser acompanhado de comentários que
   justifiquem o seu projeto orientado a objetos
3. Gere o modelo de classes da Cafeteira (ObjectAid)
4. Adicione o código, modelo e testes implementados ao repositório git:
   git add <arquivos>
   git commit -m "<uma mensagem que documente a versão>"
   git push
